## 1. 애플리케이션 메뉴 및 화면 구성
Windows 11 Fluent Design(Mica 효과) 기반의 5대 핵심 뷰로 구성됩니다.

*   **대시보드 (Dashboard)**: 현재 가동 중인 분석 공장 전체 상황 모니터링 및 워치독 제어.
*   **하베스트 (Harvest)**: 신규 작품 검색, 크롤링 및 한국어 메타데이터 세공.
*   **프로세싱 (Processing)**: 4단계 정밀 분석 엔진의 실시간 진행 상황 및 로그 관찰.
*   **라이브러리 (Library)**: 분석 완료 데이터 탐색, 인물 관계도 그래프 및 스토리 아카이브.
*   **세팅 (Settings)**: 3가지 테마(Win11/화이트/다크), API 키, 리소스 매니지먼트 설정.

## 2. 코어 파이프라인 (Stage 1~7)
기초적인 데이터 수집부터 지능형 자동화까지의 핵심 루프입니다.

*   **Stage 1: Harvest**: 하이브리드 크롤링 (Playwright + DrissionPage). **Grok 스토리 맥락 JSON**(공통 모듈 + 수집 직후 캐시) **구현 완료.**
*   **Stage 2: Refine**: 한국어 로컬라이징 및 배우명 표준화.
*   **Stage 3: Queue**: INBOX 감시 및 안정성 체크 기반 작업 대기열 구성.
*   **Stage 4: Analyze**: 4단계 정밀 분석 (Skeleton -> Signal -> VLM -> Assembly).
*   **Stage 5: Build**: 분석 결과물을 SQLite DB에 직접 연동 (Desktop GUI).
*   **Stage 6: Maintain**: 에러 복구(04_ERROR) 및 데이터 마이그레이션.
*   **Stage 7: Watchdog**: 24시간 무인 자동화 분석 루프 가동.

---

## 3. 플러스 알파 부가 기능 (Stage 1~8+)
실제 사용성과 편의성을 극대화하는 지능형 부가 도구입니다.

*   **시맨틱 장면 검색**: "비오는 날 옥상" 등의 키워드로 모든 장면 검색.
*   **스마트 점프 플레이어**: 요약 클릭 시 해당 타임라인으로 즉시 이동 재생.
*   **일괄 수정 도구**: 여러 영상의 배우/태그 정보를 한 번에 관리.
*   **심야 분석 스케줄러**: PC 미사용 시간에 분석 자원을 집중 할당.
*   **DB 무결성 체크**: 파일 경로 및 썸네일 상태 자동 복구.

## 4. 비저너리 콘텐츠 큐레이션 (Stage 9~14)
콘텐츠를 즐기는 방식을 혁신하는 AI 기반 경험들입니다.

*   **하이라이트 리얼 생성**: 영상의 정수만 뽑아낸 3분 요약본 자동 편집.
*   **개인화 취향 리포트**: 나의 스토리 선호도를 학습하여 제공하는 정기 보고서.
*   **지능형 자막 오버레이**: 자막 파일 없이도 AI 분석 내용을 화면에 실시간 노출.
*   **라이브러리 위키 수출**: Obsidian/Notion용 거미줄 구조 지식 베이스 추출.
*   **AI 포스터 아트**: 작품의 분위기를 재해석한 예술적인 디지털 포스터 생성.

---

## 5. 자가 진화 및 초개인화 (Stage 15+)
시스템 스스로 성능과 정확도를 개선하는 지각 있는 아키텍처입니다.

*   **지능형 안면 인식**: 메타데이터가 없는 영상의 배우를 얼굴로 자동 식별.
*   **시각적 유사도 히트맵**: 연출적 구도가 유사한 다른 작품의 씬 추천.
*   **능동적 피드백 학습**: 사용자의 취향을 반영하여 요약 문체와 관점 진화.
*   **Plex/Stash 연동**: 분석된 고품질 데이터를 외부 미디어 서버와 자동 동기화.
*   **AV1 스마트 압축**: 분석 완료 후 화질 손실 없이 용량을 50% 절약하는 아카이빙.

---

## 6. 최종적 특이점 및 초월적 경험 (Stage 20+)
기술과 예술이 결합된 JAVSTORY의 최종 도달점입니다.

*   **스토리 장르 변조**: 분석된 줄거리를 필름 느와르 등 원하는 톤으로 재구성.
*   **감정/긴장도 히트맵**: 전체 120분의 몰입도를 그래프로 시각화.
*   **보이스 어시스턴트**: 목소리만으로 라이브러리 필터링 및 조작.
*   **AI 영상 복원**: 저해상도 핵심 장면을 4K급 고화질로 업스케일링.
*   **3D 가상 라이브러리**: VR 공간에서 가상 갤러리 형태로 컬렉션 탐색.

## 7. UI/UX 디자인 및 로깅 전략
*   **3대 테마**: Windows 11 Native / Pretty White / Elegant Dark 지원.
*   **슬림 로깅**: UI의 미를 해치지 않는 숨겨진 로그 드로어와 진단 정보 복사 기능.
*   **리소스 Safeguard**: VRAM 사용량을 실시간 모니터링하며 순차 순환 분석 강제.
